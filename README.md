# MongoDB_Operations
Full-stack web application developed using MERN stack that features essential MongoDB operations, including inserting, finding, updating, deleting, and aggregating documents. It leverages the power of MongoDB's flexible document model and operations like insertOne, find, updateOne, deleteOne, and aggregate to manage data effectively. The frontend interacts seamlessly with the backend to provide an intuitive understanding to basic MongoDB operations.

Technologies Used: React, Express, Node.js, MongoDB
